CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir cómo se presenta un documento HTML o XML. Proporciona un conjunto de reglas y propiedades que permiten definir la apariencia visual de los elementos en una página web. La estructura básica de CSS consta de selectores, propiedades y valores. Los selectores se utilizan para seleccionar los elementos HTML a los que se aplicarán las reglas de estilo. Las propiedades definen las características específicas que se aplicarán, como el color, el tamaño de fuente, el espaciado y la posición. Los valores son los ajustes específicos que se asignan a las propiedades, como un color específico o un tamaño de fuente determinado.

HTML

HTML (Hypertext Markup Language) is a standard markup language used for creating web pages and web applications. It provides a structure and format for organizing content on the internet. HTML uses a set of tags to define the elements of a web page, such as headings, paragraphs, links, images, tables, forms, and more. HTML documents are composed of a series of elements enclosed within opening and closing tags. The tags are written using angle brackets (<>) and consist of the element name. Elements can have attributes that provide additional information or modify the behavior of the element.

REACT

React es una biblioteca de JavaScript utilizada para construir interfaces de usuario interactivas y reactivas. Fue desarrollada por Facebook y se ha vuelto muy popular en el desarrollo web moderno. React utiliza un enfoque basado en componentes, lo que significa que las aplicaciones se dividen en piezas más pequeñas y reutilizables llamadas componentes. Estos componentes encapsulan la lógica y la interfaz de usuario de una parte específica de la aplicación. React utiliza una sintaxis de JSX (JavaScript XML) que combina HTML y JavaScript, lo que facilita la creación de componentes.
